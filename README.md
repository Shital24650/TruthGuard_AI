# 📌 Overview

TruthGuard AI is an advanced AI-powered misinformation detection and digital forensics platform designed to analyze digital content for:

- Fake news
- Manipulative narratives
- Psychological influence
- Timeline inconsistencies
- Emotional bias
- Sensational framing
- Fabricated claims

Unlike traditional fake-news classifiers, TruthGuard AI focuses on **Explainable AI**, helping users understand *why* content may be misleading instead of only labeling it as true or false.

Built for the **Academia EU Hackathon 2026**, the project promotes:

- Responsible AI
- Digital awareness
- Media literacy
- Online safety
- Democratic information integrity

---

# 🚨 The Problem

Modern misinformation spreads faster than verified truth.

False narratives today often:

- Exploit fear and emotional reactions
- Mimic trusted media branding
- Blend facts with fabricated information
- Manipulate public perception
- Use AI-generated content for influence
- Spread virally through social platforms

Most existing systems only classify content without explaining their reasoning.

This creates:
- Low transparency
- Limited educational value
- Reduced public trust
- Weak digital literacy

---

# 💡 Our Solution

TruthGuard AI acts as a real-time **Digital Forensics Intelligence Engine**.

The platform performs:

✅ Credibility Scoring  
✅ Misinformation Detection  
✅ Timeline Verification  
✅ Bias Analysis  
✅ Psychological Trigger Detection  
✅ Risk Assessment  
✅ Explainable AI Reasoning  
✅ Neutral Factual Rewriting  

Instead of only detecting misinformation, the system teaches users how misinformation works.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Shital24650/TruthGuard_AI.git
```

## 2️⃣ Navigate to Project

```bash
cd TruthGuard_AI
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Configure Environment Variables

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

## 5️⃣ Run Development Server

```bash
npm run dev
```
