# 🛡️ TruthGuard AI — Veritas Analysis Engine

TruthGuard AI is a real-time misinformation detection and digital forensics system that analyzes news/content for credibility, bias, psychological manipulation, and timeline inconsistencies — providing explainable AI-driven insights and factual corrections.

---

## 🚀 Live Demo
https://truth-guard-ai-eta.vercel.app/

---

## 📌 Problem Statement

In today’s digital world, misinformation spreads faster than truth. Fake news often:
- Uses trusted branding to mislead users
- Mixes real events with fabricated claims
- Exploits emotions like fear and shock
- Manipulates public perception using AI-generated content

TruthGuard AI is built to detect, explain, and neutralize misinformation in real time.

---

## 💡 Solution

TruthGuard AI acts as a digital forensics intelligence engine that:

- 🧠 Analyzes content credibility (0–100 score)
- ⚠️ Detects misinformation and fake narratives
- 🕒 Identifies timeline inconsistencies (future/delayed events)
- 🎭 Detects psychological manipulation techniques
- 📊 Highlights biased or sensational language
- ✍️ Rewrites content into factual, neutral form
- 🔍 Explains why content is misleading (not just classification)

---

## 🧠 Key Features

### 🔴 Credibility Scoring Engine
Provides a real-time trust score (0–100) based on content reliability.

### ⚠️ Misinformation Detection
Detects fake news patterns including:
- Fake political events
- Fabricated sources
- Misleading headlines

### 🕒 Timeline Forensics
Flags impossible or future-dated claims.

### 🎭 Psychological Trigger Analysis
Detects manipulation techniques such as:
- Fear-based framing
- Authority misuse
- Shock value headlines

### 📊 Bias Detection System
Identifies political, emotional, or sensational bias.

### ✍️ Truthful Rewrite Engine
Converts misleading content into a neutral, factual version.
